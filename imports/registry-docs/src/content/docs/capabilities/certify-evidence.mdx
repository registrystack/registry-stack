Certify evidence is the claim evaluation and evidence artifact capability.
The primary implementation is Registry Witness, which evaluates configured eligibility claims
against registry data, applies disclosure policy, and returns structured results, CCCEV-shaped
JSON-LD, or SD-JWT VC credentials.

The old v1 term is Prove.
In v2 language, Prove maps to evidence certification: a service can attest that a configured claim
was evaluated and can package the result for a calling program or wallet flow.
Registry Witness does not decide program entitlement; the caller decides what a verified claim
means for a benefit, service, or workflow.

Use this capability when a system needs a yes/no predicate, a redacted result, a value-bearing
result, delegated evaluation from a trusted peer, or credential issuance without giving the caller
the records that produced the result.
