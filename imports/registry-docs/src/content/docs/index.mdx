Registry Legend documents the registry family: the projects that publish registry metadata,
serve protected registry data, and issue evidence credentials. Six projects, three product
stories ([Publish, Consult, Prove](./start/what-we-built/)).

- **Evaluating standards conformance?** Start at [Standards register](./reference/standards/).
