Registry Stack turns sensitive registry data you already hold into protected, audited, read-only
APIs, and answers evidence questions without handing over the underlying records.

Two products do the work. **Registry Relay** returns records to authorized callers over scoped,
purpose-bound read APIs. **Registry Notary** returns a yes or no answer, a selected value, or a
signed credential without exposing the full record.

These docs are how you build with both; the product pages are pulled from each repository at build
time, so they always match the code. For the pitch (why registry surfaces matter, who benefits,
where the stack fits), see the marketing site at [registrystack.org](https://registrystack.org/).

Start with [See it live](start/see-it-live/): a hosted lab, zero install, one protected read and one
signed credential in under two minutes.
