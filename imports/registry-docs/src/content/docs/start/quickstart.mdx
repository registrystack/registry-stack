Use this page when you are new to Registry Stack and need the shortest route to a useful answer.

Two products are documented on this site: Registry Relay makes existing registry-like data
available through protected, read-only APIs, and Registry Notary answers configured evidence
questions without exposing a full source record, and can issue signed credentials.
Stay in these docs for the architecture, product boundaries, API reference, and the product docs
pulled from each repository. Go to [registrystack.org](https://registrystack.org/) for the pitch:
why registry surfaces matter, who benefits, the use cases, and the wider stack beyond these two
products.
