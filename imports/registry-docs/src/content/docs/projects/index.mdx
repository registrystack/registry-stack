Each project in the registry family has its own home in Registry Legend.
Use this index when you know which product you want; use the
[Project map](../map/) when you do not.

Every project page set follows the same shape: an overview (what it is and where
it fits), one or more how-to pages (the tasks you actually do with it), and a
reference page (the surfaces it exposes). Evidence links into the upstream repo
appear inline; gaps are marked with `TODO[evidence]` notes rather than asserted.

## The five projects

### Registry Manifest

Portable metadata renderers and a CLI that compile a single YAML manifest into
catalog, DCAT, BRegDCAT-AP, SHACL, JSON Schema, OGC API Records, ODRL, and
evidence-offering artifacts.

- [Overview](./registry-manifest/)
- [Validate and render a manifest](./registry-manifest/validate-and-render/)
- [Profile fixtures](./registry-manifest/profile-fixtures/)
- [Reference](./registry-manifest/reference/)

### Registry Relay

Read-only HTTP gateway that binds protected tabular sources (CSV, XLSX, Parquet,
Postgres) to entity-shaped APIs with API-key and OIDC authorization, audit
logging, and runtime metadata routes.

- [Overview](./registry-relay/)
- [Run Registry Relay locally](./registry-relay/run-locally/)
- [Authorize callers](./registry-relay/authorize-callers/)
- [Reference](./registry-relay/reference/)

### Registry Witness

Standalone claim service that evaluates configured eligibility claims against
registry sources and issues SD-JWT VC credentials with selective disclosure.

- [Overview](./registry-witness/)
- [Run Registry Witness locally](./registry-witness/run-locally/)
- [Configure a claim](./registry-witness/configure-a-claim/)
- [Reference](./registry-witness/reference/)

### Registry Atlas

Browser workbench for inspecting what a registry publishes and checking strict
capability matches against the standards it claims to emit.

- [Overview](./registry-atlas/)
- [Run Registry Atlas locally](./registry-atlas/run-locally/)
- [Inspect a registry](./registry-atlas/inspect-a-registry/)
- [Reference](./registry-atlas/reference/)

### Registry Lab

Docker Compose demo of the registry runtime on a laptop. Wires three Relay
instances, three Witness instances, a static metadata publisher, and a narrated
demo client into a single topology.

- [Overview](./registry-lab/)
- [Run the narrated demo flow](./registry-lab/demo-flow/)
- [Reference](./registry-lab/reference/)
