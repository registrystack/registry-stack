This is a maintenance record for site contributors. If you're an integrator, ignore this page and use the current docs in each project.

Documentation for the registry stack still lives in the individual source repos.
This index classifies each entry so readers know which docs are authoritative today,
which are historical context, and which are archive or vendored content.

Status values:

- `current`: still authoritative for its topic; not yet migrated into these docs.
- `historical`: superseded or scope-replaced; keep as record, not as instructions.
- `archive`: vendored copy, pre-rename worktree content, or nested demo staging area.
