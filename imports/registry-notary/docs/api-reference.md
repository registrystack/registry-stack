# Registry Notary API Reference

> **Page type:** Reference · **Product:** Registry Notary · **Layer:** consultation, evaluation, credential, administration · **Audience:** integrator

This page is a hand-maintained snapshot. It is not the source of truth. The
authoritative route surface is the OpenAPI document generated by
`cargo run -p registry-notary-bin -- openapi`, and the authoritative set of
application problem codes lives in the server code. When this page and either
authoritative source disagree, the authoritative source wins. Once generation
tooling exists, this page should be generated from those sources rather than
edited by hand.
