# Support

Use [GitHub issues](https://github.com/registrystack/registry-stack/issues) for
non-security bugs, questions, feature discussion, and documentation problems.

Registry Stack is a pre-1.0 technical release for evaluation, integration
pilots, and public review. Public issues are reviewed according to project
priority and maintainer capacity; no support SLA is implied.

Start with:

- Product overview: <https://registrystack.org/>
- Technical docs: <https://docs.registrystack.org/>
- Hosted lab: <https://lab.registrystack.org/>
- Local demo: `cd lab && just quick`

Do not open public support requests for suspected vulnerabilities, credential
disclosure, auth bypass, audit redaction failure, source connector data leakage,
or signing key handling bugs. Follow [SECURITY.md](SECURITY.md).
